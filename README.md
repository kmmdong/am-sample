# Introduction

아파트멘터리 랜딩페이지(https://apartmentary.co.kr) 헤더 구현 

## Supported Browser

- IE, Edge
- Chrome 
- Firefox
- Safari

---
## Prerequisites

[Yarn](https://yarnpkg.com/en/docs/install)을 권장합니다.

---
## Installation & Start

 ```
 git clone https://github.com/kmmdong/am-sample
 cd am-sample
 yarn install
 yarn start
 ```
 ### wepack-dev-server 
 ```
 http://localhost:8080/ 에서 동작합니다.
 ```
## Test

```
  cd am-sample
  yarn test
```
